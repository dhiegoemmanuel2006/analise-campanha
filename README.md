# Script que faz análise de campanhas Whatsapp

## No estágio onde trabalho, é possível fazer o campanho de envios em massa por uma aplicação e diversas vezes é necessário fazer uma análise dos dados e gerar um relatório para os clientes. Reparei que os formatos costumam vir em um formato padrão e fiz um script para que seja possível medir alguns indices de perfomance da campanha.


## Fluxo

> 1. Tenta ler o Dataframe

> 2. Tira os principais indices de performance da campanha que podem ser relevante

> 3. Imprime em formato Markdown de uma forma que seja possível só Copia e Colar na conversa com o cliente/executor da campanha.
